Finance Can and Should Strongly Rely on Analyzing Data!

There are couple things how big data has revolutionzied finance

  1. Make prediction more accurate: in financial area, people like to predict. Using the big data to analyze help the prediction more accurate.
  2. Better risk management: the result from analyzing data is objective, which means it will not be affected by humans' emotions. So, it can reduce the chance of making errors due to humanity. In addition, the big data by analyzing the historical data, can tell us the possible event beforehand.

    3. And using big data will bring a lot of benefits !

    1. It can increase the revenue because of the better prediction and risk management.
    2. It runs way more faster than human do.
    3. It can help streamline things, because all we need to do with the computer are giving it data (maybe it can grab itself), and giving it the function to operate.
    4. It can help to set up economic policies, which will benefit the whole society.

Some people might argue that "nah, there are too many things involved in finance, so it is too pratical to use a computer to account for all the effects". It will be true if we don't have much data, but now we have huge amount of data that can be used to analyze finance.
